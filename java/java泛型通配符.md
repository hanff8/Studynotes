通配符缩写说明：
- E - Element (在集合中使用，因为集合中存放的是元素)
- T - Type (Java 类)
- K - Key (键)
- V - Value (值)
- N - Number (数值类型)
- R - Result (返回结果，多用于函数式编程)
	PS: 其实以上这些对于java编译器来说作用都一样，都是作为一个占位作用，要在使用时就能确定类型，也就是说A,B,C...Z等大写字母都可以表示泛型，只不过以上泛型的定义，更多是我们程序员间约定俗成的命名方式，一种规范，从而达到“见其名则知其意”的目的，罢了。
